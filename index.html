Supabase Storage Setup (Gratis)

1. Buat akun gratis di supabase.com
2. Buat project baru → Storage → buat bucket bernama cvs → set ke Public
3. Project Settings → API → salin URL dan anon key di bawah

Tanpa Supabase: CV disimpan sebagai base64 (max ~4MB/file, hanya di browser ini).
Dengan Supabase: Semua ukuran file, link download permanen, bisa diakses dari browser mana pun.
